Classes for working with emails

## Install

Via Composer

``` bash
$ composer require phlib/mail
```
or
``` JSON
"require": {
    "phlib/mail": "*"
}
```

## Usage

``` php
<?php
use Phlib\Mail;

Factory::createFromString('');
```
